# Pipeline comercial

## Etapas

1. Prospecto: oportunidad identificada, sin contacto.
2. Calificación: validación de fit comercial, monto, timing y decisores.
3. Contactado: existe gestión inicial con mandante o actor relevante.
4. Propuesta: existe oferta, cotización o alternativa técnica.
5. Negociación: ajustes comerciales o técnicos.
6. Ganado: oportunidad adjudicada o cerrada positivamente.
7. Perdido: descartada, postergada o adjudicada a tercero.

## Priorización inicial

La carga inicial crea oportunidades cuando el proyecto presenta alta relevancia o ventana comercial activa. Esta priorización es una base de trabajo, no una validación comercial definitiva.

## Campos clave

- Monto estimado.
- Probabilidad.
- Próxima acción.
- Responsable de cuenta.
- Notas de gestión.
- Etapa.
- Score de relevancia.
- Contactos disponibles según rol.

## Rutina comercial sugerida

- Revisión semanal de oportunidades accionables.
- Asignación de responsable por sector o región.
- Contacto inicial con mensaje trazable.
- Actualización de etapa y próxima acción.
- Revisión mensual de valor ponderado del pipeline.
